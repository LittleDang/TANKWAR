# TANKWAR
1. 这个项目是用来创建一个控制台版本的多人远程坦克大战
2. 练习多人协作，项目管理，远程通讯，c++能力，ai算法能力
* [socket教程](https://www.cnblogs.com/skynet/archive/2010/12/12/1903949.html)

### 第一阶段目标
实现可以创建多个对战房间，每个对战房间可以容纳多个玩家的坦克大战。
#### 逻辑算法要求
1. 每个玩家可以控制一台坦克，可以通过键盘上下左右移动，以及发射子弹。坦克可以在一定的间隔后发射第二发子弹，子弹有一定的移动速度，子弹触碰到障碍物或者飞出地图或者击中另一台坦克都要可以判断，并且作出对应的反馈。
2. 可以载入地图，地图是以文件的形式事先存储好的，存储好障碍物的位置和种类信息。
3. 可以生成一定数量的ai坦克，ai坦克的约束和玩家操控的坦克的约束一样，唯一不一样的是不用操作。

#### 渲染要求
1. 可以实时显示坦克和地图信息（控制台）
2. 可以显示系统提示信息（控制台）

#### 网络通讯要求
1. 可以同时处理多个房间和多个玩家的数据
2. 有一个套登陆和验证系统
3. 保持同一个房间里面的玩家的数据一致

#### 程序
1. 服务器
2. 客户端

#### 运行环境
服务器/客户端 ubuntu16.04 64位
